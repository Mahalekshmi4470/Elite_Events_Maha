# 🎉 Elite Events — Event Management Website

A premium event management website for **Elite Events**, a Chennai-based company specializing in weddings, corporate events, and birthday parties.

## 📁 Project Structure

```
elite-events/
├── index.html          # Home page
├── about.html          # About Us page
├── wedding.html        # Wedding Events service page
├── corporate.html      # Corporate Events service page
├── birthday.html       # Birthday Parties service page
├── gallery.html        # Photo Gallery
├── pricing.html        # Pricing & Packages
├── faq.html            # Frequently Asked Questions
├── blog.html           # Blog page
├── team.html           # Our Team page
├── contact.html        # Contact Us page
├── sitemap.xml         # SEO Sitemap
├── robots.txt          # Search engine crawl rules
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   ├── script.js       # Main JavaScript
│   └── schema.js       # SEO Structured Data (JSON-LD)
└── images/             # All image assets (50 files)
```

## 🚀 Features
- Responsive design with mobile hamburger menu
- SEO optimized with meta tags, Open Graph, Twitter Cards
- Schema.org structured data (Organization, FAQ, Services)
- Google Fonts (Poppins) + Font Awesome icons
- Interactive FAQ accordion
- Newsletter subscription popup
- WhatsApp floating button
- Gallery with lazy-loaded images
